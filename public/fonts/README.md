# Font Files

Place variable font files here:
- `dm-sans-variable.woff2`
- `rethink-sans-variable.woff2`

## Download fonts:

### DM Sans
https://fonts.google.com/specimen/DM+Sans
Download as variable font (woff2)

### Rethink Sans  
https://fonts.google.com/specimen/Rethink+Sans
Download as variable font (woff2)

## Tools:
- Use https://google-webfonts-helper.herokuapp.com/fonts for easy downloads
- Or download directly from Google Fonts and convert to woff2
